# Secure Medical Report System

## Features
- User login with OTP verification
- Secure file upload (AES encryption)
- Role-based access (Doctor/Patient)
- Audit logging system

## Tech Stack
- Python (Flask)
- MySQL
- Cryptography (AES)

## Setup
1. Create .env file
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py

## Note
Sensitive data is stored in environment variables for security.
